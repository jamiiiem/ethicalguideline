Section A: Privacy and Data Protection

AI chatbots that deal regularly with educational content should handle them and interact with users with the highest standards of privacy and data security. This section lists the specific goals that need to be achieved in order to ensure that the privacy and data protection is upheld to a reasonable standard. These goals include data anonymisation, data minimisation, informed consent, secure data storage and transmission, user control, compliance with laws, as well as privacy audits.

A2 Data Minimisation

Definition:

Data collected and retained should be minimised to relevant and essential data.

Techniques:

  • Data Collection Policies: avoid excessive data collection by only gathering user input, necessary contextual data, and avoid collecting data that is not directly relevant to fulfil the purpose of the chatbot.
  • Data Retention Policies: set clear data retention policies that define how long data will be stored before automatic deletion.
  • Data Purging: implement processes to regularly delete old, unused data.

Examples:

Automatic data purging processes may be used to regularly delete data that has not been used for a set period of time.
