E7 Transparent Error Handling

Definition:

Developers should be open about errors or mistakes made by the chatbot and how they are addressed.

Techniques:

  • Error Disclosure: Ensure the chatbot openly acknowledges when it makes an error, explaining what went wrong.
  • Correction Mechanisms: Implement mechanisms that allow users to see how errors are corrected and what steps are taken to prevent recurrence.
  • User Communication: Proactively communicate with users if errors affect them, explaining the situation and any actions they need to take.
